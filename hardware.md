# PN532 to Raspberry Pi 3 Model B Wiring

| PN532 | Raspberry Pi |
| ----- | ------------ |
| 3.3V  |   3.3V (1)   |
|  SCK  |   SCLK (23)  |
| MISO  |   MISO (21)  |
| MOSI  |   MOSI (19)  |
| SSEL  |    CE0 (24)  |
|  GND  |    GND (25)  |
